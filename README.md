## 🚀 Getting Started

You can run this project locally in two simple ways.

### Option 1: Open Directly
Open `index.html` in any modern web browser.

### Option 2: Run Using a Local Server (Recommended)

**Using Python**
```bash
python -m http.server 8000
```
Then visit:
```
http://localhost:8000
```

**Using Node.js**
```bash
npx serve .
```
